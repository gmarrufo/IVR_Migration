Installation (Note: If the source code changes, execute README-RAD.txt before exporting the new jar file)
1. Create a folder to hold the files e.g. f:\migration
2. Copy ini.zip and EUSMigration.jar to f:\migration
3. Install java runtime environment version 1.6 or higher
4. Download http://www.antlr.org/download/antlr-3.2.jar to f:\migration

Configuration
1. Right click on EUSMigration.jar and select Open With->Java SE Binary
2. The f:/migration/migration.properties default configuration file is created
3. Edit this file. Most importantly specify which ini file you want to migrate
4. Set developmentMode to false for running as a line command. 

Execution 
1. Right click on EUSMigration.jar and select Open With->Java SE Binary
2. By default, the results are placed in the f:/migration/CallRoutingConfig

Execution with RAD (Recommended)
See README-RAD.txt
